# Landing Page Components

Beautiful, animated landing page for CouponAI with smooth Framer Motion animations.

## Components

- **LandingHeader** - Fixed header with navigation and CTA buttons
- **HeroSection** - Main hero with headline, stats, and primary CTA
- **FeaturesSection** - 6 feature cards with icons and hover effects
- **HowItWorksSection** - 4-step process explanation
- **CTASection** - Final call-to-action with benefits list
- **LandingFooter** - Footer with links and social media

## Features

- Smooth scroll animations with Framer Motion
- Gradient backgrounds and hover effects
- Responsive design (mobile, tablet, desktop)
- Modern glassmorphism effects
- Optimized performance with viewport-based animations

## Usage

The landing page is automatically rendered at the root route `/`.
